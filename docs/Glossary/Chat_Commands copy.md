# Terms

## UUID

UUIDs are 128-bit numbers. In this case, they are used
to represent player IDs. For example:

Display Name: Sample Player
User name: Player
UUID: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

In place of the x's, numbers and small case letters
are assigned to each Brickadia account individually.
